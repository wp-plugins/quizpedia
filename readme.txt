Embed QuizPedia.com quizzes easily on your blog

== Description ==

Embed quizzes from QuizPedia.com on your blog

== Installation ==

1. Upload `quizpedia.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I embed both quizzes and cards =

Yes. In the settings panel you can choose if you want to embed quizzes fully or simply as a card that links to the quiz
